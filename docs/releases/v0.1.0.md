# 桌面清理大师 v0.1.0

首个公开版本。它是一个 macOS 桌面清理玩笑软件：点击“一键清理”后，桌面看起来会变干净，但文件不会被删除、重命名或移动到别处。

## 这版包含什么

- 一键把 Finder 桌面图标临时收纳到窗口投影内。
- 关闭、退出或手动还原时恢复图标位置和 Finder 桌面设置。
- 窗口拖动时图标跟随，图标很多时自动降频并在松手后全量同步。
- 支持简体中文、繁體中文、English、日本語、한국어 菜单与界面文案。
- 打包文件名统一为 `桌面清理大师.app.zip`。

## 下载和运行

下载 `桌面清理大师.app.zip`，解压后打开 `桌面清理大师.app`。

如果 macOS 提示来自网络下载，可先清除 quarantine 标记：

```bash
xattr -dr com.apple.quarantine "桌面清理大师.app"
open "桌面清理大师.app"
```

## 权限说明

首次真实清理时，macOS 会要求允许应用控制 Finder。这个权限用于读取和恢复桌面图标位置。

## 已知限制

- 仅支持 macOS。
- 当前是 ad-hoc signed build，没有 Developer ID 签名，也没有 notarization。
- 公开分发时，用户可能仍需要手动处理 macOS 安全提示。
- 本工具只调整 Finder 桌面图标显示位置，不处理真实文件内容。
