# 桌面清理大师 v1.1.0

## 这版包含什么

- 英文展示名调整为 `Desktop Cleaner`，避免 `Desktop Cleanup Master` 的生硬营销感。
- 新增 Español、Français、Português、Deutsch、हिन्दी 本地化，并保留 English、简体中文、日本語、繁體中文、한국어。
- App 界面、菜单、完成态、错误提示、性能提示和 macOS 自动化权限弹窗都使用本地化文案。
- 本地化语义按语言调整：中文和欧美语言使用“桌面图标 / desktop icons”等自然表达，日语、韩语和 Hindi 使用更适合本地语境的“項目 / 항목 / आइटम”。
- 日语和韩语主标题加入显式断行，避免系统自动换行拆开词组。

## 下载和运行

下载 release 附件 `Desktop-Cleaner.zip`，解压后打开 `桌面清理大师.app`。

如果 macOS 提示来自网络下载，可先清除 quarantine 标记：

```bash
xattr -dr com.apple.quarantine "桌面清理大师.app"
open "桌面清理大师.app"
```

## 权限说明

首次真实清理时，macOS 会要求允许应用控制 Finder。这个权限用于读取和恢复桌面图标位置。

## 已知限制

- 仅支持 macOS。
- 当前是 ad-hoc signed build，没有 Developer ID 签名，也没有 notarization。
- 公开分发时，用户可能仍需要手动处理 macOS 安全提示。
- 本工具只调整 Finder 桌面图标显示位置，不处理真实文件内容。
